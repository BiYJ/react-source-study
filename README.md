# React源码学习
本仓库在 [react-source-code-debug](https://github.com/neroneroffy/react-source-code-debug)的基础上保留了V17和Lanes模型，且删除掉了大量无关文件、代码（如其中的__DEV__相关判断)，只保留了核心的react、react-dom、react-reconciler、和scheduler

## 安装依赖
npm install

## 启动React不同版本的调试环境：
* 启动17正式版
```
 npm run dev:17
```
* 启动Lanes版本
```
npm run dev:lanes
```
